# Contribute to MediaBrowser

MediaBrowser is open to all contributions! You just need to respect the code of conduct.

## How to contribute?

You can contribute to MediaBrowser several ways:
- Software
- Documentation
- Translation

### Software

What you need to contribute to MediaBrowser:
1. Visual Studio 2022 with UWP .NET Desktop.
2. .NET SDK 6 / 7
3. (Optionaly) At least a Xbox One / Xbox Series with Dev Mode activated. See https://www.microsoft.com/en-us/p/xbox-dev-mode/9nljhzjrn0f4

To contribute:
1. Fork this repository
2. On your forked repository, create a new branch for developping your work.
3. Do modification.
4. Always test your modification.
5. If all goods, send a Pull Request to your master branch on your forked repository.
6. Send a Pull Request to this repository and follow instructions.

If you don't have a Xbox, please specificate on the PR.

### Documentation

You can fork this repository to make change on the documentation.
